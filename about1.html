Programming

In CSOC, we practice programming in many languages, though the most common are:

  • Python - A versatile language used for web development, data analysis, artificial intelligence, and more.
  • JavaScript - A scripting language primarily used for creating interactive effects within web browsers.
  • HTML/CSS - The standard markup language and styling sheets used for creating and designing web pages.

Projects

Some good projects for learning how to code:

  • CS50 Courses - Free online courses that cover the basics of programming, computer science, and web development using Python and JavaScript.
  • High Seas - A program for students under the age of 19 to make their own projects, reveiw other projects and get free stuff.
  • Web Development - Create your own website or web app using HTML, CSS, and JavaScript.
